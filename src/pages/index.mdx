# Frames

Start with a frame and specify the sides that should be visible:

```
<Frame></Frame>
```

<Frame></Frame>

Width is 100% by default and height as small as needed, just like a `<div>` element. You can add optional `width` and `height` attributes, a custom `className` or `style` to adjust this. You can also specify which borders to show (`t`, `l`, `r`, `b` for top, left, right, bottom). Content goes inside the frame:

```
<Frame t l={150} b={4} height={400}>
    Content here
</Frame>
```

<Frame t l={150} b={4} height={400}>
    Content here
</Frame>

To add buttons to a side of the frame, use the specialized `<Left>`, `<Top>`, `<Right>` and `<Bottom>` elements together with `<Button>` and `<Filler>` elements

```
<Frame l t b height={400}>
    <Left>
        <Button>Engage</Button>
        <Filler flex />
    </Left>
    content here
</Frame>
```

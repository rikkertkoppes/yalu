Use `<Row>` and `<Col>` together with `<Filler>`, `<Button>`, `<Gap>` and `<Cap>` elements to create button rows and columns. You can also use the `flex` attribute to make the row or column fill the available space.

```
<Col width={150} height={200}>
    <Row>
        <Button flex>Engage</Button>
        <Cap />
    </Row>
    <Row>
        <Button flex>Disengage</Button>
        <Cap />
    </Row>
    <Row flex>
        <Filler flex>95-468</Filler>
        <Cap />
    </Row>
</Col>
```

Rows can also be used for generic layouting

<Row className="lcars-gray" height="10">
    <Filler flex>87-268</Filler>
    <Filler flex></Filler>
</Row>
